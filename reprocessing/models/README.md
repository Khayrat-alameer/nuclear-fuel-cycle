# Reprocessing Models

This directory contains models for spent nuclear fuel reprocessing, including PUREX process, separation chemistry, and recovery efficiency.

## Model Types

- **Separation chemistry models**: Solvent extraction and chemical separation processes
- **PUREX process models**: Plutonium Uranium Reduction Extraction simulation
- **Recovery efficiency models**: Actinide and fission product recovery rates
- **Waste stream models**: Secondary waste generation and characterization

## File Naming Convention

- `separation_chemistry_*.py`: Chemical separation process models
- `purex_process_*.py`: PUREX cycle simulation models
- `recovery_efficiency_*.py`: Material recovery rate models
- `waste_stream_*.py`: Secondary waste generation models
