# Reactor Operation Models

This directory contains models for in-reactor fuel behavior, neutronics, thermal hydraulics, and fuel performance during reactor operation.

## Model Types

- **Neutronics models**: Neutron transport, multiplication, and flux distribution
- **Thermal-hydraulic models**: Heat transfer, coolant flow, and temperature distribution
- **Fuel performance models**: Fission gas release, swelling, and mechanical behavior
- **Burnup models**: Isotopic evolution and reactivity changes over time

## File Naming Convention

- `neutronics_*.py`: Neutron transport and reactor physics models
- `thermal_hydraulic_*.py`: Heat transfer and fluid flow models
- `fuel_performance_*.py`: In-reactor fuel behavior models
- `burnup_analysis_*.py`: Isotopic evolution and depletion models
