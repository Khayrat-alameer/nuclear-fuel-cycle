# Reactor Operation Module

## Quick Start

```bash
streamlit run reactor_operation/web_app.py
```

## Features

- Neutronics: k-effective, flux, burnup
- Thermal Hydraulics: heat transfer, DNBR
- Fuel Performance: fission gas release, stress

## Models

- `neutronics.py`: Reactor physics
- `thermal_hydraulic.py`: Heat transfer
