# Fabrication Models

This directory contains models for nuclear fuel fabrication processes, from powder preparation to final fuel assembly.

## Model Types

- **Powder processing models**: UO2 powder characteristics and behavior
- **Pellet fabrication models**: Pressing, sintering, and quality control
- **Fuel rod assembly models**: Cladding, welding, and assembly processes
- **Quality assurance models**: Defect detection and performance prediction

## File Naming Convention

- `powder_processing_*.py`: Powder characterization and handling models
- `pellet_fabrication_*.py`: Pellet manufacturing process models
- `rod_assembly_*.py`: Fuel rod assembly and welding models
- `quality_assurance_*.py`: Quality control and defect analysis models
