# Enrichment Models

This directory contains models for uranium enrichment processes, including centrifuge cascades, diffusion, and laser enrichment technologies.

## Model Types

- **Cascade models**: Multi-stage centrifuge cascade simulation
- **Separation efficiency models**: Isotope separation process modeling
- **Energy consumption models**: Power requirements and optimization
- **Material flow models**: Uranium hexafluoride (UF6) handling and conversion

## File Naming Convention

- `cascade_simulation_*.py`: Centrifuge cascade models
- `separation_efficiency_*.py`: Isotope separation performance models
- `energy_consumption_*.py`: Power usage and optimization models
- `material_flow_*.py`: UF6 flow and conversion models
