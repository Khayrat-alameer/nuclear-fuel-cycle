# Mining Models

This directory contains mathematical and computational models related to uranium mining and extraction processes.

## Model Types

- **Resource estimation models**: Geological modeling and resource quantification
- **Extraction efficiency models**: Leaching and recovery process modeling
- **Environmental impact models**: Mining footprint and remediation modeling
- **Economic models**: Cost-benefit analysis and resource optimization

## File Naming Convention

- `resource_estimation_*.py`: Geological resource estimation models
- `extraction_process_*.py`: Chemical and physical extraction models
- `environmental_impact_*.py`: Environmental assessment models
- `economic_analysis_*.py`: Economic evaluation models
