# Waste Disposal Models

This directory contains models for long-term nuclear waste management, geological disposal, and safety assessment.

## Model Types

- **Geological repository models**: Repository design and performance assessment
- **Radionuclide transport models**: Migration through geological barriers
- **Engineered barrier models**: Canister corrosion and buffer material behavior
- **Safety assessment models**: Long-term risk and dose calculation

## File Naming Convention

- `repository_design_*.py`: Geological repository performance models
- `radionuclide_transport_*.py`: Migration and transport models
- `engineered_barrier_*.py`: Barrier system performance models
- `safety_assessment_*.py`: Risk and dose assessment models
